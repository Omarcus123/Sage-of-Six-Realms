About Me

Yooo! My name is Omarcus "PureFGC" Swims Jr. I'm a passionate fighting game enthusiast, with a particular love for Mortal Kombat at the moment. My goal is to create a hub for players like myself who want to dive deep into character mechanics, frame data, and strategies.

I graduated with a degree in Computer Science, which allows me to combine my love for gaming with my technical skills to build this website. The inspiration for this project came from the player RedRaptor and their Kombat Akademdy website, which set a standard for fighting game resources.

Currently, this site provides detailed information on character frame data and how to get started playing mortal kombat 1 with a little bit of naruto embedded into it. In the future, I plan to expand it with combo guides, more in depth character guides, and other tools to help both newcomers and seasoned players improve their game. My hope is to create a community where players can learn, share, and level up together.

Thanks for visiting, and I hope you enjoy exploring the site as much as I enjoy building it!
